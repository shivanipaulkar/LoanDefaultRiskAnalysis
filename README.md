Loan Default Risk Analysis

Project Objective
The objective of this project is to analyze borrower characteristics and identify factors contributing to loan defaults using data analysis techniques. The project uses Python, SQL, Excel and Power BI to extract insights and build an interactive dashboard for credit risk analysis.

Tools & Technologies
1. Python (Pandas, NumPy, Matplotlib, Seaborn)
2. SQL Server
3. Power BI
4. Excel (Initial Data Exploration)

Dataset Description: The dataset contains 255,347 loan records with borrower attributes such as:Age, Income, Loan Amount, Credit Score, Employment Type, Loan Purpose, Debt-to-Income Ratio (DTI), Default Status

Project Workflow
1. Data Understanding
2. Data Cleaning & Feature Engineering (Python)
3. Exploratory Data Analysis (EDA)
4. Business Query Analysis using SQL
5. Power BI Dashboard Development
6. Insights & Business Recommendations

Business Questions Answered
1. What is the overall loan default rate?
2. How does income level affect default risk?
3. How does credit score influence loan default?
4. Which age groups are more likely to default?
5. Does employment type impact default probability?
6. Does loan size influence default risk?

Key Insights
1. Low income borrowers show significantly higher default rates.
2. Borrowers with poor credit scores contribute most to loan defaults.
3. Younger borrowers (18–30) have higher default probability.
4. Larger loan amounts are associated with increased risk.
5. Full-time employment reduces default likelihood.

Business Recommendations
1. Apply stricter credit evaluation for low-income applicants.
2. Use credit score thresholds to reduce high-risk lending.
3. Monitor large loan amounts more closely.
4. Provide financial education programs for young borrowers.

Dashboard

An interactive Power BI dashboard was created to visualize default risk across borrower segments.

Key dashboard features include:

Loan default rate KPIs
Risk analysis by income, credit score, and loan size
Interactive slicers for borrower segmentation

Project Outcome

This project demonstrates the use of Excel for initial exploration, Python for data analysis, SQL for querying, and Power BI for visualization to support data-driven decision-making in credit risk analysis.
